Hỗ Trợ & Hướng Dẫn Sử Dụng

Nơi giải đáp mọi thắc mắc và giúp bạn cấu hình dịch truyện bằng các API AI tiên tiến nhất hiện nay.

🔑 Hướng dẫn cấu hình API Key

  1. Truy cập trang web Google AI Studio tại: https://aistudio.google.com/.
  2. Đăng nhập bằng tài khoản Google của bạn.
  3. Nếu đây là lần đầu truy cập, hãy đồng ý với các điều khoản điều kiện sử dụng dịch vụ.
  4. Tại giao diện chính, nhấp vào nút "Get API key" ở góc trên bên trái (hoặc thanh menu bên trái).
  5. Chọn "Create API key".
  6. Chọn dự án (project) hiện có hoặc tạo mới, sau đó chọn "Create API key in existing project".
  7. Sao chép (Copy) khóa API (API Key) vừa được tạo.
  8. Mở ứng dụng Novara, vào mục Cài đặt và dán (Paste) API Key vào ô cấu hình Google Gemini.

💡 Hạn mức miễn phí: API Key Gemini miễn phí có giới hạn tần suất yêu cầu (RPM - Requests Per Minute), nhưng hoàn toàn đáp ứng tốt nhu cầu dịch truyện cá nhân của bạn mà không phát sinh chi phí.

💡 Mẹo nhỏ: Mỗi tài khoản Google có thể tạo tối đa 10 project tương ứng với 10 API key khác nhau, giúp bạn dịch và đọc truyện cả ngày mà không lo về việc bị gián đoạn hay hết key.

⚠️ Cảnh báo bảo mật: Tuyệt đối KHÔNG chia sẻ API Key của bạn lên Internet. Nếu bị lộ, người khác có thể sử dụng hết hạn mức (quota) của bạn.

  1. Truy cập trang đăng ký/đăng nhập của DeepSeek tại: https://platform.deepseek.com/sign_in.
  2. Đăng nhập bằng tài khoản DeepSeek của bạn (hoặc đăng ký nếu chưa có tài khoản).
  3. Đi đến mục "API Keys" ở thanh menu quản lý bên trái.
  4. Nhấp vào nút "Create new API key".
  5. Nhập tên gợi nhớ (ví dụ: "Novara Translator") rồi nhấp "Create".
  6. Sao chép (Copy) khóa API vừa được tạo (Lưu ý: Khóa chỉ hiển thị một lần duy nhất tại bước này).
  7. Mở ứng dụng Novara, vào mục Cài đặt và dán (Paste) API Key vào ô cấu hình DeepSeek.

💡 Cơ chế thanh toán: DeepSeek hoạt động theo cơ chế nạp tiền trước sử dụng sau (Pay-as-you-go). Bạn cần nạp một số tiền nhỏ vào tài khoản (mục Top Up) để kích hoạt API.

💡 Ưu điểm: Giá dịch vụ dịch thuật của DeepSeek cực kỳ rẻ và chất lượng dịch văn phong truyện Trung Quốc rất mượt mà, tự nhiên.

⚠️ Cảnh báo bảo mật: KHÔNG chia sẻ API Key của bạn với bất kỳ ai để tránh phát sinh chi phí ngoài ý muốn trong tài khoản của bạn.

❓ Câu hỏi thường gặp (FAQ)

Làm sao để bắt đầu dịch truyện?

▼
Mở ứng dụng → Nhấn "Thêm EPUB" → Chọn file EPUB trên thiết bị của bạn → Cấu hình và bắt đầu quá trình dịch.

Tôi cần gì để sử dụng ứng dụng?

▼
Bạn cần chuẩn bị một API key từ Google Gemini (miễn phí) hoặc DeepSeek (có phí). Sau đó vào Cài đặt trong ứng dụng → nhập API key → bắt đầu dịch truyện.

Dữ liệu sách có được upload lên server không?

▼
Không. Toàn bộ dữ liệu sách được lưu cục bộ trên thiết bị của bạn. Chỉ văn bản cần dịch mới được gửi trực tiếp đến API AI (Google/DeepSeek) thông qua API key của bạn để xử lý.

Tôi có thể dịch tối đa bao nhiêu cuốn sách?

▼
Không giới hạn số lượng sách. Chi phí dịch phụ thuộc hoàn toàn vào quota API key của bạn (đối với gói miễn phí của Gemini thì hoàn toàn không tốn tiền).

Ứng dụng hỗ trợ những ngôn ngữ dịch nào?

▼
Hiện tại, Novara tập trung tối ưu sâu cho việc dịch từ Tiếng Trung (giản thể / phồn thể) sang Tiếng Việt.
