Q: Làm sao để bắt đầu dịch truyện?

Mở ứng dụng → Nhấn "Thêm EPUB" → Chọn file EPUB trên thiết bị → Cấu hình và bắt đầu dịch.

Q: Tôi cần gì để sử dụng?

Bạn cần một API key từ Google Gemini (miễn phí) hoặc DeepSeek. Vào Cài đặt → nhập API key → bắt đầu dịch.

Q: Dữ liệu sách có được upload lên server không?

Không. Toàn bộ dữ liệu sách được lưu cục bộ trên thiết bị. Chỉ văn bản cần dịch mới được gửi trực tiếp đến API AI (Google/DeepSeek) thông qua API key của bạn.

Q: Tôi có thể dịch bao nhiêu sách?

Không giới hạn số lượng sách. Chi phí dịch phụ thuộc vào quota API key của bạn.

Q: Ứng dụng hỗ trợ ngôn ngữ nào?

Hiện tại Novara tập trung vào việc dịch từ Tiếng Trung (giản thể/phồn thể) sang Tiếng Việt.
